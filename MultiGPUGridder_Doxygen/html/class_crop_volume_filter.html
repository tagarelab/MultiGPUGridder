Detailed Description

A filter for cropping a GPU array.

CropVolumeFilter inherits from the AbstractFilter class.

The CropVolumeFilter crops a GPU array. This is needed for removing the zero padding after running the forward or inverse Fourier transform operations.

In order to set which GPU the filter should be run on, please use the cudaDeviceSet() function before calling the Update function.
