Detailed Description

A filter for converting a cufftComplex GPU array to CAS type.

ComplexToCASFilter inherits from the AbstractFilter class.

This class converts a complex GPU array to CAS type. The gpuForwardProject and gpuBackProject classes need the arrays to be CAS type. The main advantage of using CAS for the projection is reduced memory size and faster computation speed.

In order to set which GPU the filter should be run on, please use the cudaDeviceSet() function before calling the Update function.
