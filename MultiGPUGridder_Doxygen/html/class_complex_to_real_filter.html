Detailed Description

A filter for extracting the real component of a cufftComplex array.

ComplexToRealFilter inherits from the AbstractFilter class.

This class extracts the real component of a cufftComplex array on the GPU. This is needed for obtaining the output of the forward and inverse Fourier transform operations.

In order to set which GPU the filter should be run on, please use the cudaDeviceSet() function before calling the Update function.
