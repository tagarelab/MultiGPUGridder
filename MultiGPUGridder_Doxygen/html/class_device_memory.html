Detailed Description

template<class T = float>
class DeviceMemory< T >

A class for allocating device (i.e. GPU) memory.

A class for allocating and deallocating GPU memory. DeviceMemory also remembers needed information for each allocated array (e.g. CASImgs, images, coordinate axes, etc.) such as the array size, memory pointers, etc. This is the main GPU memory class.

DeviceMemory inherits from HostMemory and extends DeviceMemory to include GPU related information and functions.
